HTML MainTags

  1. <sectionTag> in section tag two tags are defined in sectionTag.they are: <headerTag> <footerTag>
  2. if we want to use header tag and footer tag then we must use <sectionTag>
  3. headerTag:-content will appear on bold letters in webpage.
  4. footerTag:-content will apper on small letters in webpage.
  5. articleTag:-is used to represent blogspot,newspaper,article
  6. mainTag:-is used to display the main content in the webpage
  7. divTag:-if we want to apply css and create some extra styles then it is used.
