HTML Syntax

  1. An HTML element usually consists of a start tag and end tag, with the content inserted in between:Content goes here....
  2. All HTML documents must start with a document type declaration:.
  3. The HTML document itself begins with and ends with .
  4. Head tag represents the document's header and its start with and close with .
  5. The title tag is used inside the head tag to mention the document title.
